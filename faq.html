¿Por que no me llegan las vidas?

Para poder obtener 3 vidas diarias tienes que hacer una recarga mayor o igual a 10 soles cada mes. Tambien puedes comprar paquetes de vidas para mas información selecciona el menú (Ofertas de paquetes de vidas). Al entrar a la aplicación diariamente automáticamente se entregan las vidas (aveces puede tardar unos minutos en aparecer las vidas en tu cuenta)

¿Por que me redirige a la app Mi Bitel?

La aplicación Mi Bitel se abrirá una vez cada día para reclamar las vidas luego retornará automáticamente al Cuy Mágico, ya que el juego requiere que ingreses diariamente para que te otorgue las 3 vidas diarias.

¿por qué el tiempo de espera después de cada premio?

El tiempo de espera es impuesto por el servidor de bitel, si este se reduce no recibes ningún premio y pierdes vidas, si vas a abrir muchas puertas se recomienda ajustar el tiempo mayor a 4 segundos para evitar restricciones del operador.
